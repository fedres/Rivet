Complete Feature Set

📦

Zero-Config Dependencies

Add libraries with one command via vcpkg integration. Automatic download, configuration, and linking.

rivet add boost
🏗️

Workspace Support

Multi-package monorepo builds with shared target directory and dependency consolidation.

rivet build # workspace
📚

Library Compilation

Static (.a) and shared (.so) library generation with ar integration and proper linking.

crate-type = "staticlib"
🎯

Multiple Target Types

Binaries, libraries, examples, and benchmarks in one project with automatic build coordination.

[[bin]], [lib], [[example]]
🌍

Cross-Compilation

Target triple management with automatic vcpkg triplet mapping for multiple platforms.

rivet target add x86_64-pc-linux
⚡

Fast Commands

Quick syntax checking, artifact cleanup, and build-and-run workflows for rapid development.

rivet check && rivet run

Project Management

  • rivet init - Initialize in current directory
  • rivet new <name> - Create new project

Build System

  • rivet build - Build project/workspace
  • rivet check - Fast syntax checking
  • rivet clean - Remove build artifacts
  • rivet run [--bin <name>] - Build and execute

Dependencies

  • rivet add <package> - Add dependency
  • rivet remove <package> - Remove dependency

Toolchains

  • rivet toolchain list - List toolchains
  • rivet toolchain install <name> - Install toolchain
  • rivet toolchain use <name> - Set active toolchain

Configuration Examples

Simple Binary

[package]
name = "my_app"
version = "0.1.0"

[dependencies]
fmt = "*"
boost = "1.82"

Library Package

[package]
name = "mylib"
version = "0.1.0"

[lib]
name = "mylib"
crate-type = "staticlib"  # or "dylib"

Workspace (Monorepo)

[package]
name = "my_workspace"
version = "0.1.0"

[workspace]
members = [
    "core",
    "utils",
    "app",
]
