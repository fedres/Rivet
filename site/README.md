# Rivet GitHub Pages Website

This is the official website for Rivet - a modern C++ project manager inspired by Cargo.

## Features Documented

- **21 Commands** - Complete command reference
- **Workspace Support** - Multi-package monorepo builds
- **Library Compilation** - Static and shared libraries
- **Cross-Compilation** - Platform target management
- **vcpkg Integration** - Automatic dependency management

## Website Structure

- `index.html` - Main landing page with features
- `docs.md` - Complete documentation
- `styles.css` - Website styling
- `script.js` - Interactive elements
- `404.html` - Custom error page

## Local Development

```bash
# Serve locally
python3 -m http.server 8000

# Visit
open http://localhost:8000
```
